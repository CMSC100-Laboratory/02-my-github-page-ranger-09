**Steps on how to create GitHub page:**  
1. Clone your repository
2. Create an html file then push it into your repository.
3. Visit GitHub.com
4. Go to your repository.
5. Go to settings.
6. Go to pages.
7. Under the branch section, change "None" to "main"
8. Wait a few minutes then refresh your screen, a url link will appear.
